# GreenTech-MVP-Sostenible_Alejandro_MorenoLuna
Lo primero que hice, fue separar el HTML del CSS, ya que cuando el CSS es interno el navegador tiene que descargarlo cada vez que el usuario visita la  pagina.
Además, mejoraría el rendimiento y haría todo mucho mas rápido y ameno.
Por otro lado, he eliminado el Bootstrap ya que para el proyecto que tenemos, consumía mucha energía, y memoria por los servidores. Por ello, al usar únicamente el CSS nativo fomentamos el uso de la pagina web ya que dispositivos de gama baja podrían usarla perfectamente.
A su vez, implementamos un Hover para saber si el cursor está encima del botón o no.
Por último, he cambiado el formato de la imagen tras buscar en internet como hacerlo para hacer el CSS y el HTML lo mas simple posible con el objetivo de que la imagen ocupe un alto porcentaje menos.
